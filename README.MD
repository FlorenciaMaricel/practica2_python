Este proyecto contiene ejercicios en Python sobre procesamiento de texto, estructuras de datos y lógica de programación.

Instalación y ejecución:

- Clonar el repositorio:
  git clone https://github.com/FlorenciaMaricel/practica2_python.git

  cd practica2_python

- Crear entorno virtual:
  python -m venv venv

- Activar entorno virtual:
  Bash/Git Bash/Linux/Mac: source venv/Scripts/activate
  CMD Windows: venv\Scripts\activate.bat
  PowerShell Windows: .venv\Scripts\Activate.ps1

- Instalar dependencias:
  pip install -r requirements.txt

- Abrir Jupyter Lab:
  jupyter lab
  Esto abrirá el navegador con los notebooks dentro de la carpeta notebooks/.

Estructura del proyecto:

- notebooks/: notebooks con los ejercicios
- src/: código fuente (.py)
- .git/: control de versiones
- requirements.txt: dependencias
- README.md: este archivo
